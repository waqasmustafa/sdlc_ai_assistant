About Odoo AI Assistant

A powerful AI Assistant built natively for Odoo 18 that lets anyone query live business data using natural language. Ask questions in plain English and get instant, formatted answers from your CRM leads, sales orders, HR records, inventory levels, accounting data, contacts, products, and more — all without writing SQL or building custom reports.

The module uses a 3-step AI pipeline (Prompt Refiner → Query Generator → Response Formatter) powered by Groq Cloud, supports 6 AI models with auto-fallback, and respects Odoo’s ORM access rights so each user only sees data they’re authorized to view.

⚙ How It Works

3-Step AI Pipeline

Ask a question → AI fetches your data → Get a formatted answer in seconds

1

You Ask

Type your question in plain English. The Prompt Refiner cleans and optimizes it for the AI model.
2

AI Processes

The Query Generator builds the ORM call. Data is fetched securely through Odoo access-rights.
3

Get Answer

The Response Formatter returns a clean answer with real data from your Odoo instance.

⚡ AI Models

6 AI Models at Your Fingertips

Switch instantly. Auto-fallback ensures zero downtime.

Role-Based Access Control

Configure in Settings → Users → Access Rights. Three levels control who chats and who configures.

No Access

AI Assistant is completely hidden. Cannot see or access the app.

User: Chat Only

Chat with AI, view own history. No configuration access.

Admin: Full Access

Chat + configure AI settings, models, intents, users.

Getting Started in 4 Steps

Up and running in under 2 minutes. Install, configure, and start querying.

1

Open AI Assistant

Click “AI Assistant” in the Odoo sidebar. The chat interface opens with conversation history.
2

Select AI Model

Llama 3.3 70B (Smart) recommended for accuracy. Switch anytime from the dropdown.
3

Ask Your Question

Type in plain English, press Enter. Try: “How many contacts?”, “Top 5 sales”
4

Get Your Answer

Formatted answer with real data. Export as CSV, JSON, or download conversations.
